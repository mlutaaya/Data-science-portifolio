Project: Worldwide Peace¶

About Project¶

Inter-language translation is invaluable for travelers in foreign countries. With translation apps, they can easily translate menus, road signs, and more. These apps typically utilize services like Google Translate. In this project, I explore the range of languages supported by Google Translate and create a script to translate the English word "Peace" into each of these languages. Finally, I present the translations in a word cloud, with all text displayed in the same size.

Translating the English word "Peace" into each of the supported languages

In [5]:
translator = Translator() # loading translator
others = [language for language in dictionary.keys() if not (language == 'en')] #excluding english from destination langaues
translations =   [(translator.translate('Peace', dest = x).text) for x in others] # translation in action!!
